# 📖 为什么会有这个项目？

收藏越来越多之后，你是不是也遇到过这些问题？

- 🤔 这个谷子我到底有没有买？
- 💳 补款是不是快截止了？
- 📦 还有哪些快递没有收到？
- 🎁 这个系列还差哪几个？
- 💰 当初是多少钱收的？
- 🗂 收藏放在哪个柜子来着？

很多时候，答案散落在聊天记录、订单截图、备忘录和 Excel 里。

真正想找的时候，却总要翻很久。

于是，就有了 **谷子收纳**。

希望把所有收藏，都整理到同一个地方。
