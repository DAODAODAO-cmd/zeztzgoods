# 📦 谷子收纳

*A local collection manager for goods collectors.*

> **「买的时候很快乐，整理的时候很崩溃。」**
> 所以干脆做一个属于自己的谷子收纳网页。

---

# 为什么做这个？

一开始真的只是想记一下：

> **"我到底买了什么？"**

结果收着收着……

开始变成：

* 这个是不是有两块？
* 这个是在哪个平台收的？
* 当时到底多少钱？
* 邮费有没有算进去？
* 为什么这个月又花了这么多？🤔

于是，这个网页就慢慢长出来了。

它没有什么伟大的目标。

只是希望以后打开的时候，可以很轻松地知道：

> **「哦，原来我真的有这个。」**

---

# 能做什么？

目前已经可以：

✅ 收藏 / 出物管理

✅ 图鉴模式（美美欣赏）

✅ 表格模式（认真整理）

✅ 圆图模式（强迫症福音）

✅ 图片上传

✅ 搜索、筛选

✅ 收藏评级

✅ 价格趋势

✅ 自动统计花费

✅ 自动计算盈亏

✅ 导入 / 导出

✅ 自动保存

基本上每天收谷、整理、出物都会用到。

---

# 一个网页，收所有坑

以前想着：

> **"就做这个 IP 吧。"**

后来发现……

根本不可能。

今天收这个。

明天掉进另一个坑。

后天又发现：

> **"诶？这个也挺好看。"**

于是现在支持：

📦 系列 IP

📦 多作品

每个作品都拥有独立的：

* 角色
* 系列
* 种类
* 渠道
* 获取方式
* 状态
* Logo
* 背景
* 字段配置

以后无论是：

* 假面骑士
* 动漫
* 游戏
* VTuber
* 偶像
* 还是其他任何作品

都可以直接建立自己的收藏库。

毕竟……

坑可以越来越多。

网页还是一个就够了。🌚

---

# 完全可以自己改

网页里的东西几乎都不是写死的。

角色、系列、种类、渠道、获取方式……

全部可以：

✅ 新增

✅ 修改

✅ 删除

甚至连：

* Logo
* 背景
* 字段名称
* 系列 IP
* 作品

都可以自己调整。

理论上……

除了钱包。

其他几乎都能改。💸

---

# 背景也可以换

终于不用一直看同一个背景了。

现在支持：

🖼 上传自己喜欢的图片作为背景。

无论是：

* 官方宣传图
* 收藏展示图
* 自己拍的照片
* 喜欢的壁纸

都可以变成属于自己的收藏主页。

毕竟……

天天看谷子。

背景当然也要一起美美的。

---

# 氪金统计

这个功能其实有一点危险。

因为它会非常诚实地告诉你：

**你到底氪了多少钱。**

有时候打开统计页面：

> "……"

> "这个数字是真的吗？"

> "算了，还是关掉吧。"

当然。

第二天还是会继续买。

统计存在的意义，

不是劝你别买。

而是提醒你：

**至少知道自己花去哪了。😂**

---

# 本地保存

所有数据默认保存在浏览器本地。

不会强制上传。

不会绑定账号。

不用担心突然哪天打不开网站，

收藏就跟着一起消失。

不过……

## **真的记得导出备份！**

浏览器要是被清掉。

你的谷子不会消失。

但是记录会。

到时候只能：

> "我记得我有……"

> "但是放哪了？"

---

# 市场数据（开发中）

未来准备加入：

📈 匿名市场数据统计。

大家贡献一点点。

一起慢慢拼出整个市场。

以后可以看到：

* 平均成交价
* 热门商品
* 平台价格对比
* 建议挂牌价
* 市场价格趋势

上传的都是**非敏感数据**。

不会上传：

* 图片
* 备注
* 联系方式
* 任何可以识别个人身份的信息

希望以后。

不仅知道：

> **"我买贵了吗？"**

还能知道：

> **"现在卖是不是最佳时机？"**

---

# 最后

这个网页不是为了做成什么大型系统。

它只是一个陪着收藏一起成长的小工具。

希望以后每次打开，

不是为了计算自己又花了多少钱。

而是慢慢翻着自己的图鉴。

看着这些一路收集来的收藏。

然后笑着说一句：

> **「嗯，又多了一件。」**

……

当然。

如果统计金额能再低一点。

如果新品能少出一点。

如果钱包能再厚一点。

那就更好了。🌚

---

> **记录，也是收藏的一部分。**
