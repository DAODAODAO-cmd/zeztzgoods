# 仮面ライダーゼッツ 谷子收纳

*A local collection manager for goods collectors.*

> **「买的时候很快乐，整理的时候很崩溃。」**
> 所以干脆做一个属于自己的谷子收纳网页。

---

# 为什么做这个？

一开始真的只是想记一下：

> **"我到底买了什么？"**

结果收着收着……

开始变成：

* 这个是不是有两块？
* 这个是在哪个平台收的？
* 当时到底多少钱？
* 邮费有没有算进去？
* 为什么这个月又花了这么多？🤔

于是，这个网页就慢慢长出来了。

它没有什么伟大的目标。

只是希望以后打开的时候，可以很轻松地知道：

> **「哦，原来我真的有这个。」**

---

# 能做什么？

目前已经可以：

✅ 收藏 / 出物管理

✅ 图鉴模式（美美欣赏）

✅ 表格模式（认真整理）

✅ 圆图模式（强迫症福音）

✅ 图片上传

✅ 搜索、筛选

✅ 收藏评级

✅ 自动统计花费

✅ 自动计算盈亏

✅ 导入 / 导出

✅ 自动保存

基本上每天收谷、整理、出物都会用到。

---

# 完全可以自己改

网页里的分类几乎都不是写死的。

角色、系列、种类、渠道、获取方式……

都可以自己新增、修改、删除。

虽然现在是按照 **仮面ライダーゼッツ** 做的，

但如果哪天突然掉坑别的 IP……

改一下分类，它马上就能继续上班。😂

---

# 氪金统计

这个功能其实有一点危险。

因为它会非常诚实地告诉你：

**你到底氪了多少钱。**

有时候打开统计页面：

> "……"

> "这个数字是真的吗？"

> "算了，还是关掉吧。"

当然。

第二天还是会继续买。

---

# 本地保存

所有数据都会保存在浏览器本地。

不会上传。

不会联网。

不会偷偷同步。

所以也意味着——

## **记得导出备份！**

真的。

非常重要。

如果浏览器资料被清掉，没有备份的话，

收藏资料就只能一起进入梦境了。🥲

---

# 最后

这个网页不是为了做成什么大型系统。

它只是一个陪着收藏一起成长的小工具。

希望以后每次打开，

不是为了计算自己又花了多少钱，

而是可以慢慢翻着自己的图鉴，

然后很满足地说一句：

> **「嗯，收得真不错。」**

……当然。

如果统计金额能再低一点，那就更好了。🌚
